# 热点Top10筛选与评分（可复制执行）

## 评分维度（0–5分）

对每个“事件级”候选热点打分并排序：

1. **时效性**
   - 5：12小时内爆发/持续更新
   - 3：24小时内
   - 1：更早但重新发酵
2. **覆盖度（信源数量）**
   - 5：≥5个独立权威媒体/平台
   - 3：2–4个
   - 1：单一信源
3. **讨论热度（外显信号）**
   - 5：出现“热搜/置顶/多平台同时上榜”等信号
   - 3：多篇跟进报道
   - 1：普通新闻
4. **内容可写性**
   - 5：能给观点/科普/实用建议/反转点
   - 3：可做信息整合
   - 1：只有事实通报
5. **合规风险（反向项）**
   - 5：低风险
   - 3：中性但需措辞
   - 1：明显高风险（隐私、血腥、涉敏）→ 建议淘汰

**总分建议：**前4项相加后减去风险惩罚：
- `总分 = 时效性 + 覆盖度 + 讨论热度 + 可写性 - (5 - 合规风险)`

## 去重归并规则

- 同一事件的不同标题：以“人物/地点/机构 + 核心动作 + 时间”归并
- 同一事件不同阶段：若出现明显新进展（例如官方通报/判决/重大反转）可拆成两条

## 输出规范（方便后续写小红书）

每条Top10必须输出：
- `事件标题（≤18字）`
- `一句话概括（≤28字）`
- `看点（1句）`
- `建议角度（科普/观点/实用/情绪）`
- `来源链接（1–2条）`
